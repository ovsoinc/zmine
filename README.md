### Run Client
```bash
git clone https://github.com/super3/zmine
cd ~/zmine
npm install
npm start
```

### Run Server
```bash
git clone https://github.com/super3/zmine
cd ~/zmine
npm install pm2 -g
pm2 start process.json
```
